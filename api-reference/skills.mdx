---
title: "Skills API — list and run AI analyses"
description: "List available AI skills and run them against a client — from Meta ad analysis to SEO audits, lead scoring, outbound sequences, and reports."
---

Skills are VERA's AI analysis units. Each skill takes a client and a set of inputs, calls Claude with enriched context, and returns structured output. All skills share the same run interface — `POST /api/v1/skills/{skill_id}/run`.

---

## Get a skill definition

<br />

`GET /api/v1/skills/{skill_id}`

Returns the full definition object for a single skill.

### Path parameters

<ParamField path="skill_id" type="string" required>
  Skill ID from the table above (e.g. `meta-ads-analysis`).
</ParamField>

```bash
curl https://api.verahq.ai/api/v1/skills/meta-ads-analysis \
  -H "X-API-Key: YOUR_API_KEY"
```

## Run a skill

<br />

`POST /api/v1/skills/{skill_id}/run`

Runs a skill against a client. The client must belong to your authenticated workspace.

**Rate limit:** 20 calls per minute per API key.

### Path parameters

<ParamField path="skill_id" type="string" required>
  Skill ID to run.
</ParamField>

### Body parameters

<ParamField body="client_id" type="string" required>
  UUID of the VERA client to run the skill for.
</ParamField>

<ParamField body="inputs" type="object">
  Skill-specific input parameters. The required keys vary by skill — refer to the skill's definition for the expected shape. An empty object `{}` is valid when no inputs are required.
</ParamField>

### Response fields

<ResponseField name="skill_id" type="string">The skill that ran.</ResponseField>
<ResponseField name="client_id" type="string">The client the skill ran against.</ResponseField>
<ResponseField name="status" type="string">`completed` or `failed`.</ResponseField>
<ResponseField name="output" type="object">Structured output from the skill. Shape varies by skill.</ResponseField>
<ResponseField name="cost_usd" type="number">LLM cost for this run in USD.</ResponseField>
<ResponseField name="model" type="string">Claude model used.</ResponseField>
<ResponseField name="input_tokens" type="integer">Prompt tokens consumed.</ResponseField>
<ResponseField name="output_tokens" type="integer">Completion tokens produced.</ResponseField>
<ResponseField name="started_at" type="string">ISO 8601 start timestamp.</ResponseField>
<ResponseField name="completed_at" type="string">ISO 8601 completion timestamp.</ResponseField>

<Tabs>
  <Tab title="Request">
    ```bash
    curl -X POST https://api.verahq.ai/api/v1/skills/meta-ads-analysis/run \
      -H "X-API-Key: YOUR_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "client_id": "clt_9f2e1a",
        "inputs": {}
      }'
    ```
  </Tab>

## Get skill run history

<br />

`GET /api/v1/skills/runs/{client_id}`

Returns all skill runs for a client, newest first.

### Path parameters

<ParamField path="client_id" type="string" required>
  The client's UUID.
</ParamField>

<Tabs>
  <Tab title="Request">
    ```bash
    curl https://api.verahq.ai/api/v1/skills/runs/clt_9f2e1a \
      -H "X-API-Key: YOUR_API_KEY"
    ```
  </Tab>
  <Tab title="Response">

      "runs": [

          "started_at": "2026-04-19T10:30:00+00:00"

    }

</Tabs>
